# 📝 NotebookApp — веб-приложение для быстрого создания заметок

> Проект выполнен в рамках дисциплины «Введение в направление» студентами 1 курса программной инженерии Челябинского Государственного Университета.  
> Приложение позволяет создавать текстовые заметки и списки, добавлять теги, выбирать цвет фона, закреплять важное, фильтровать и сортировать.  
> Все данные хранятся локально в браузере (`localStorage`) — сервер не требуется.

## ✅ Функциональные возможности

### Базовый CRUD
- Создание заметки (текст или список с чекбоксами)
- Редактирование заметки (все поля)
- Удаление заметки
- Чтение – все заметки отображаются в виде списка

### Дополнительные атрибуты
- Закрепление (пин) — закреплённые заметки поднимаются вверх
- Назначение тегов
- Выбор цвета фона

### Управление просмотром
- Фильтрация по выбранному тегу 
- Сброс фильтра 
- Сортировка по дате создания и по приоритету

### UI/UX требования
- Карточка заметки содержит: превью текста/списка, теги, цвет, кнопки действий
- Длинный текст обрезается с многоточием
- При наведении на карточку появляются кнопки (редактировать, запинить)
- Адаптивная вёрстка: на мобильных устройствах карточки в одну колонку, кнопки удобны для касания
- Пустое состояние: дружелюбное сообщение и иллюстрация, если заметок нет
- Возможность переключать на тёмную тему
